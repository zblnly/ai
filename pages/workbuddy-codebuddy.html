🧑‍💻 WorkBuddy 与 CodeBuddy 完全指南

腾讯推出的AI办公与编程助手 | 一句话完成复杂任务

什么是 WorkBuddy？

WorkBuddy 是腾讯推出的全场景职场AI智能体桌面工作台。您只需用一句话描述需求，WorkBuddy便能像同事一样自主规划和执行任务，并交付可验收的结果。

核心能力

  • 🗣️ 理解自然语言：用一句话下达任务，无需复杂的操作步骤
  • 📋 自主规划执行：自动拆解任务、规划步骤、执行操作
  • 📊 多模态任务处理：支持文档、表格、PPT、数据分析等多种任务类型
  • 📁 本地文件操作：可读取授权的电脑文件夹，进行批量处理

适用场景

工作模式 - 职场通用场景

  • 📝 文档生成：生成工作报告、技术文档、会议纪要等
  • 📈 数据分析：上传数据文件，自动分析和可视化
  • 📊 PPT/报告生成：从需求描述生成演示文稿和报告
  • 🔍 深度研究：对复杂问题进行深度调研并输出报告
  • 📧 邮件编辑：邮件撰写、周报生成等办公场景
  • 📂 批量文件处理：自动整理、重命名、转换文件格式

编程模式 - 代码开发场景

  • 💻 日常编码辅助：代码生成、代码审查、Bug修复、重构优化
  • 📖 项目理解：快速了解陌生代码库的架构和逻辑
  • 🌐 网站/应用开发：从需求到代码的全流程辅助

🚀 快速开始

1. 下载安装

访问 CodeBuddy官网 下载 WorkBuddy 或 CodeBuddy。

2. 一句话下达任务

# 示例任务 "帮我把这份销售数据生成一个分析报告" "整理Downloads文件夹里的图片，按日期分类" "根据这份会议纪要生成一个PPT" "分析这份简历，提取关键信息生成表格" "调研一下2024年AI行业趋势，写一份报告"

3. 添加上下文

  • 引用文件：通过@引用文件、文档、规则等信息
  • 粘贴截图：使用Ctrl/Cmd + V直接粘贴剪贴板中的图片
  • 上传文件：点击上传按钮或拖拽文件到输入框
  • 补充说明：在任务描述中明确目标、范围、约束和预期输出

⚡ 自动化任务配置

WorkBuddy 支持创建自动化任务，帮你完成重复性工作：

配置步骤

配置项 说明
任务名称 区分不同自动化任务
工作空间 指定任务执行目录
提示词 描述任务目标和输出要求
定时规则 设置执行周期

示例：每日AI新闻推送

任务名称：每日AI新闻简报 提示词：调研今天的AI行业动态，整理成简报， 包含标题、摘要和来源链接 工作目录：行业新闻文件夹 定时规则：每天 08:00

预设模板

内置多种模板：竞品分析、周报生成、会议纪要整理等，导入即可使用。

🔧 常见问题解决

1. 消息无响应

  • 检查网络连接是否稳定
  • 尝试切换模型
  • 简化任务描述，避免过长

2. 无法读取文件

  • 切换到支持图片/文档输入的模型
  • 分步骤描述需求
  • 安装文档处理类插件

3. 运行缓慢

  • 减少一次性过大的复合任务
  • 拆分为多个独立任务
  • 切换到更快的模型

4. 新手入门高频场景

  • 📱 控制微信，自动发朋友圈或统计群聊信息
  • 📝 自动编写并发布小红书内容
  • 🕷️ 自动采集指定网页信息并整理成文件

WorkBuddy vs 传统AI对话

对比 传统AI对话 WorkBuddy

使用技巧

任务描述公式

目标 + 输入 + 输出格式 + 约束条件 # 示例 目标：生成销售分析报告 输入：基于这份Excel数据 输出格式：包含图表的Word报告 约束条件：突出显示Top3产品

多轮对话

WorkBuddy会记住之前的对话上下文，可以：

  • 继续追问："再分析一下环比变化"
  • 调整需求："把字体调大一点"
  • 补充说明："报告加上执行摘要"

文件处理
