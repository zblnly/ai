🚀 阶段一：入门阶段（1-2周）

目标

了解AI辅助编程概念，掌握基础Prompt技巧，能够使用在线AI服务完成简单编程任务。

学习内容

  • AI辅助编程基础

    了解什么是AI编程，AI能做什么、不能做什么

  • Prompt工程入门

    学习编写有效的提示词，包括：角色设定、任务描述、输出格式约束

    → OpenAI Prompt指南
  • 在线AI服务体验

    注册 ChatGPT/Claude/Gemini 账号，体验AI代码生成

  • 基础编程任务

    让AI帮助：代码注释、函数文档、简单函数实现、bug修复

实践任务

  • 使用ChatGPT生成一个TODO应用
  • 让Claude帮你写一个API接口文档
  • 用AI解释一段你不熟悉的代码

🛠️ 阶段二：工具配置（2-3周）

目标

完成本地开发环境配置，能够在IDE中流畅使用AI辅助编程工具。

学习内容

  • 本地大模型部署

    安装Ollama，下载 qwen2.5-coder 或其他代码模型

    → Ollama配置指南
  • VSCode + Continue.dev

    安装 Continue.dev 扩展，配置本地Ollama模型

    → Continue.dev官网
  • Cursor编辑器

    下载并体验 Cursor，熟悉 AI 补全、对话、编辑功能

    → Cursor官网
  • OpenCode开源助手

    体验最强开源AI编程助手，支持75+模型

    → OpenCode配置指南

IDE推荐配置

使用场景 推荐工具 推荐模型 特点
日常开发 Cursor Claude 4 / GPT-4 开箱即用体验好
VSCode用户 Continue.dev qwen2.5-coder 深度集成生态丰富
隐私敏感 OpenCode 本地模型 数据本地处理
免费使用 CodeFuse 专有模型 免费中文优化

实践任务

  • 在VSCode中配置Continue.dev
  • 用Ollama运行本地模型
  • 用AI补全功能完成一个项目模块

⚔️ 阶段三：进阶实战（3-4周）

目标

熟练使用AI辅助编程，显著提升开发效率，能够独立完成中等复杂度项目。

学习内容

  • 高效Prompt技巧

    学习：CRISPE提示框架、思维链提示、上下文管理

  • 项目级AI应用

    用AI完成：数据库设计、API设计、测试用例编写

  • 代码审查与优化

    让AI审查代码、提出优化建议、重构代码

  • 调试与问题排查

    用AI分析错误日志、定位bug、提供修复方案

  • 文档自动化

    用AI自动生成API文档、README、代码注释

进阶技巧

# 高效Prompt示例：让AI帮你设计API 你是一位资深后端架构师。请帮我设计一个用户管理系统的RESTful API： 1. 包含功能：用户注册、登录、信息查询、信息修改、密码重置 2. 使用 Node.js + Express + MySQL 3. 请输出：API路径设计、请求参数、响应格式、数据库表结构 4. 代码要求符合最佳实践，包含错误处理
# 调试Prompt示例：让AI帮你定位bug 我遇到了一个Node.js运行时错误，错误信息如下： [Error: Cannot read property 'name' of undefined at UserService.getProfile (user.service.js:45)] 这是相关代码片段（第40-50行）： [代码片段] 请分析可能的原因，并提供修复方案。

实践任务

  • 用AI设计并实现一个完整的小项目
  • 让AI帮你重构一个旧项目
  • 用AI完成单元测试编写（覆盖率>80%）

🤖 阶段四：高级应用（持续学习）

目标

构建自动化工作流，使用AI Agent完成复杂任务，形成个人AI开发工作流。

学习内容

  • AI Agent原理

    了解Agent架构：感知→思考→行动→反馈循环

    → OpenHarness AI框架
  • 工作流自动化

    使用AI自动完成：需求分析→代码实现→测试→文档全流程

  • 多模型协作

    组合使用：GPT写代码 + Claude审查 + 本地模型处理隐私数据

  • RAG知识库

    构建私有知识库，让AI基于你的文档回答问题

  • 模型微调

    在自己的代码数据集上微调专属模型

高级工具

工具 用途 学习难度
OpenHarness AI Agent开发 中
LangChain LLM应用开发 高
AutoGPT 自主AI Agent 中
LlamaIndex RAG知识库 中

实践任务

  • 构建基于私有文档的问答系统
  • 实现一个自动化代码审查工作流
  • 尝试模型微调训练
