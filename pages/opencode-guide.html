🔧 OpenCode 完全指南

2026 年最强开源 AI 编程助手 | 140K+ GitHub Stars

什么是 OpenCode？

OpenCode 是一个开源的 AI 编码代理，帮助你在终端、IDE 或桌面端编写代码。它不仅仅是一个聊天机器人，而是一个可以执行命令、搜索代码、调试错误和修改文件的智能编程伙伴。

核心特性

  • 📱 多平台：终端、桌面应用、IDE 扩展
  • 🔒 隐私优先：不存储任何代码或上下文数据
  • 🔗 会话分享：可分享会话链接用于参考或调试
  • 🔀 多会话管理：可同时在同一项目上运行多个 Agent
  • 🤖 多模型支持：支持 75+ LLM 提供商（Claude、GPT、Gemini、本地模型等）
  • 🔌 LSP 集成：自动加载正确的语言服务器协议支持
  • 🖥️ 原生终端界面：响应快，支持自定义主题

快速开始

1. 安装 OpenCode（推荐方式）

# 一键安装（推荐） curl -fsSL https://opencode.ai/install | bash # 使用 npm npm install -g opencode-ai # 使用 Homebrew（macOS/Linux） brew install anomalyco/tap/opencode # 使用 Scoop（Windows） scoop install opencode # 使用 Arch Linux sudo pacman -S opencode

2. 下载桌面应用（可选）

访问 OpenCode 下载页面 下载原生桌面应用。

配置 AI 模型

方式 A：使用 OpenCode Zen（推荐）

Zen 是 OpenCode 团队精选并测试过的模型列表，专为编码 Agent 优化。

# 运行连接命令 opencode auth login # 选择 opencode # 访问 opencode.ai/auth # 登录并添加账单信息 # 复制 API 密钥并粘贴

方式 B：使用 Google Gemini（免费）

前往 Google AI Studio 获取免费 API 密钥（250请求/天）。

# 在 OpenCode 中配置 /opencode # 输入 /connect 选择 gemini # 粘贴你的 API 密钥

方式 C：使用 Ollama 本地模型

配合 Ollama 本地模型，可以实现真正的零成本 AI 编程。

# 先确保 Ollama 正在运行 ollama serve # 配置 opencode 使用本地模型 /opencode # 输入 /connect 选择 ollama # 设置 apiBase 为 http://localhost:11434

初始化项目

首次使用建议先运行 /init 命令，让 OpenCode 分析你的项目，自动生成一个 AGENTS.md 文件。

/init # 这将让 OpenCode 分析你的项目结构 # 并创建 AGENTS.md 文件 # 你应该将此文件提交到 Git

基本使用

两种模式切换

  • Plan 模式（只读）：按 Tab 键切换，只做分析和规划，不会动你的代码
  • Build 模式（默认）：全权限，可以读写文件、执行命令

常用命令

# 提问 - 让 AI 解释代码 How is authentication handled in @packages/api/src/index.ts # 添加功能 - 让 AI 添加新功能 Add a user profile page with form validation # 制定计划 - 只读模式 /new feature: Add dark mode toggle # 查看可用命令 /help # 切换模型 /models # 切换主题 /themes # 查看会话历史 opencode sessions list # 撤销修改 /undo

VSCode 中使用

OpenCode 可以集成到 VSCode 中使用：

  1. 在 VSCode 中打开终端
  2. 运行 opencode
  3. 按 Cmd+Esc（Mac）或 Ctrl+Esc（Win/Linux）在分屏中打开

注意事项

  • ⚠️ 2026年限制：Anthropic 在 2026 年 1 月阻止了 OpenCode 使用 Claude 模型
  • 建议使用 GPT-4o 或 Gemini 2.5 Pro 作为替代
  • 建议将 AGENTS.md 文件提交到 Git，帮助 AI 理解项目规范
  • 首次使用建议使用 /init 初始化项目
