2026年最火爆的开源自演进AI Agent - 会学习的智能助手

什么是 Hermes Agent？

Hermes Agent 是由 Nous Research 开发的开源自演进AI Agent，2026年2月发布后在6周内获得57,000+ GitHub Stars，成为增长最快的AI Agent项目。

与传统AI助手不同，Hermes Agent具有内置学习循环——它会从使用中学习，创建可复用技能，记住你的偏好，并随着使用时间推移变得越来越聪明。

核心特点

🧠 自演进学习

完成任务后自动创建新技能，在使用中不断改进，记住跨会话的上下文

💾 持久记忆

多级记忆系统，跨会话持久化，搜索历史对话，构建用户画像

🌐 多平台支持

Telegram、Discord、Slack、WhatsApp、Signal、Email、CLI

🔌 模型 agnostic

支持200+模型 via Nous Portal/OpenRouter，也支持本地模型

🚀 快速开始

安装 (一行命令)

curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

支持 Linux、macOS、WSL2、Android (Termux)

基本命令

hermes # Start interactive CLI hermes --help # Show help hermes model # Switch model hermes skills # Browse skills

配置模型

# 使用 Nous Portal (推荐，首个API key) hermes portal # 使用 OpenRouter hermes openrouter # 使用本地 Ollama hermes ollama # 使用 OpenAI hermes openai

⚙️ 配置文件

首次运行后会在 ~/.hermes/ 目录创建配置文件：

~/.hermes/.env 配置

# 至少需要一个API provider # 方式1: Nous Portal (推荐) HERMES_PORTAL_KEY=your_portal_key # 方式2: OpenRouter OPENROUTER_API_KEY=your_key # 方式3: OpenAI OPENAI_API_KEY=your_key # 方式4: Anthropic ANTHROPIC_API_KEY=your_key

~/.hermes/config.yaml 高级配置

# 持久化配置 memory: type: postgres # sqlite / postgres connection: postgresql://user:pass@localhost/hermes # 网关配置 gateway: telegram: enabled: true discord: enabled: false token: your_token # 模型配置 model: provider: portal model: hermes-3-thought-7b

🔧 基础用法

1. 命令行对话

$ hermes ☤ Starting Hermes Agent... Enter your message (or /exit to quit): 分析这个Python代码的性能问题 [Hermes正在分析...]

2. Telegram机器人

# 在 .env 中配置 TELEGRAM_BOT_TOKEN=your_bot_token # 启动后给机器人发送消息即可对话

3. 设置技能

# 查看内置技能 hermes skills list # 安装社区技能 hermes skills install agent-skills/code-review # 创建自定义技能 hermes skills create my-skill

🎯 进阶技巧

自演进系统

工作原理

  1. 观察: Agent观察任务执行过程
  2. 计划: 分析如何改进
  3. 行动: 创建/优化技能
  4. 学习: 记住学到的知识

每次完成复杂任务后，Hermes会自动分析并创建可复用的技能，下次遇到类似任务时可以直接使用。

记忆系统

多级记忆

  • 工作记忆: 当前会话的上下文
  • 情景记忆: 跨会话的重要信息
  • 语义记忆: 用户偏好和工作风格
  • ���序记忆: 学到的技能

技能系统

内置40+技能

  • 📁 文件管理 - 创建、读取、编辑、搜索文件
  • 🌐 网页浏览 - 搜索、爬取网页内容
  • 💻 代码执行 - 运行Python、JS、Shell命令
  • 🔌 API调用 - REST API封装
  • 📊 数据库 - SQL查询和管理
  • 📧 邮件 - IMAP/SMTP邮件收发

🐛 常见问题

Q: API调用失败怎么办？
A: 检查.env中的API Key是否正确，尝试切换provider: hermes model openai
Q: Telegram连接失败？
A: 确保bot token有正确权限，通过 @BotFather 创建
Q: 技能不工作？
A: 运行 hermes skills list 查看已安装技能，确保技能已启用
Q: 如何升级？
A: hermes update 或重新运行安装脚本
Q: 支持中文吗？
A: 支持，设置 LLM 模型为支持中文的模型即可
