🔧 CodeFuse 安装指南

蚂蚁集团推出的 AI 编程助手安装配置教程

什么是 CodeFuse？

CodeFuse 是蚂蚁集团推出的 AI 编程助手，基于百亿参数的代码大模型，可根据开发者的输入提供智能建议和实时支持，帮助开发者自动生成代码、增加注释、生成测试用例、修复和优化代码等。

CodeFuse 支持 10 款 IDE，包括：

  • Visual Studio Code
  • IntelliJ IDEA
  • PyCharm
  • WebStorm
  • GoLand
  • CLion
  • DataGrip
  • PhpStorm
  • RubyMine
  • 支付宝小程序云云端研发

安装前提

以 VS Code 为例，需要满足以下条件：

  • VS Code 版本 1.75.0 及以上
  • 稳定的网络连接
  • 支付宝账号（用于登录和申请试用）

安装步骤

  1. 下载插件安装包

    访问 CodeFuse 官网，下载对应 IDE 的插件安装包。目前提供的是 Beta 版本。

  2. 安装插件

    以 VS Code 为例：

    • 打开 VS Code
    • 在左侧导航栏点击扩展图标
    • 点击右上角的 "..." 图标
    • 选择 "Install from VSIX..."
    • 选中下载的 CodeFuse-x.x.x.vsix 文件
    • 点击 "Install" 按钮
  3. 登录并申请试用
    • 安装完成后，在 IDE 左侧导航栏点击 CodeFuse 图标
    • 点击 "登录" 按钮
    • 在弹窗中点击 "Open"
    • 使用支付宝登录 CodeFuse 官网
    • 填写申请理由，阅读并勾选用户服务协议和隐私协议
    • 点击 "提交申请"
  4. 等待审核

    申请提交后，需要等待审核通过。审核通过后，插件面板会显示可使用状态。

功能介绍

AI 代码补全

基于海量数据提供实时地代码补全服务，包括行内补全（单行补全）和片段补全（多行补全）。支持 Java、Python、TypeScript、JavaScript、Go 等 5 种主流编程语言的多行和单行代码补全，以及其他 40 种编程语言的单行代码补全。

AI 添加注释

智能生成代码注释，尤其在函数级别的注释生成方面表现出色，大幅提升代码的可读性和可维护性。

AI 解释代码

智能解析选中代码的意图，自动生成解释，帮助开发者更轻松地阅读和理解代码内容。

AI 生成单测

完成业务逻辑后，自动为目标代码生成单元测试，确保测试用例符合业务语义，从而更高效地发现潜在问题。

AI 代码优化

借助大模型的代码理解能力与静态分析技术，CodeFuse 可分析选定代码片段，提供优化与改进建议，并基于这些建议生成相应的代码补丁。

使用方法

代码补全

CodeFuse 支持自动代码补全和手动代码补全两种触发方式：

  • 自动触发：在编写代码时自动弹出补全建议
  • 手动触发：使用快捷键 Alt/Option + \ 手动触发代码补全

支持切换多个补全结果，最多支持切换 2 个结果：

  • Windows：Alt + ] 显示下一个补全结果，Alt + [ 显示上一个补全结果
  • Mac：Option + ] 显示下一个补全结果，Option + [ 显示上一个补全结果

右键菜单功能

选中代码后，右键菜单中可使用以下功能：

  • 添加注释
  • 解释代码
  • 生成单测
  • 代码优化

注意事项

  • 目前 CodeFuse 处于 Beta 测试阶段，可能存在稳定性问题
  • 所有智能服务均运行于云端，需保持稳定网络连接
  • 企业级部署时需考虑数据安全和隐私保护
  • 申请试用需要等待审核，审核时间可能会有延迟
