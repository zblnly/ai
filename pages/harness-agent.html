🤔 什么是Agent Harness？

先看一个公式：

Agent = Model + Harness

模型本身只是"脑子"，会思考但没手没脚。要让它帮你改代码、查文件、跑脚本、上网搜索……就得给它装上一整套工具链和安全边界。

Harness就是这套"马具"——让大模型变成真正能干活的黑衣宰相。

模型 vs Harness 的分工

部分 负责 例子
Model（模型） 思考、推理、决策 理解需求、规划步骤、选择工具
Harness（框架） 执行、安全、记忆、边界 调用工具、检查权限、保存上下文

🏗️ Harness的核心组成

🔧 Tools（工具）

让模型能够读写文件、搜索、执行命令、访问网页、调用MCP等。OpenHarness提供43+工具。

📚 Knowledge（知识）

通过skills、CLAUDE.md、MEMORY.md等机制，把外部知识装载进系统。按需加载。

👁️ Observation（观察）

让模型看到git diff、错误日志、文件状态等执行反馈，不只是"想"，还能"看"。

⚡ Action（行动）

执行工具调用、文件操作、命令执行等实际行动，而不是只返回文本。

🛡️ Permissions（权限）

安全边界：执行前权限检查、敏感操作确认、危险命令拦截。

🧠 Memory（记忆）

长期运行时的状态保存、会话恢复、上下文压缩、断点续聊。

👥 Multi-Agent（多智能体）

主Agent派发子任务、团队协调、后台任务管理、并行执行。

🐣 OpenHarness 介绍

OpenHarness 是香港大学数据科学研究院（HKUDS）开源的轻量级AI Agent框架。

  • 📅 2026年4月1日发布
  • 📝 纯Python 3.10+ 实现
  • 📊 仅 11,733 行代码（Claude Code的1/44）
  • ⭐ GitHub 5K+ Stars
  • 📜 MIT开源协议

与Claude Code对比

对比项 OpenHarness Claude Code
代码规模 11,733行 512,000+行
编程语言 Python TypeScript
开源 ✅ MIT ❌ 闭源
多模型 ✅ 支持 ❌ 仅Claude

⚡ OpenHarness 快速开始

1. 一键安装

# Linux/macOS/WSL curl -fsSL https://raw.githubusercontent.com/HKUDS/OpenHarness/main/scripts/install.sh | bash # 或者用pip pip install openharness-ai

2. 配置模型

# 交互式配置（推荐） oh setup # 或者手动设置环境变量 export ANTHROPIC_API_KEY=sk-... export ANTHROPIC_MODEL=claude-3.5-sonnet

支持：Claude / OpenAI / Codex / Moonshot(Kimi) / 通义千问 / GLM 等

3. 启动使用

# 交互模式 oh # 单次执行 oh -p "解释这个代码库" # JSON输出 oh -p "列出main.py中的函数" --output-format json

🎯 OpenHarness 核心功能

🔄 Agent Loop

流式工具调用、指数退避重试、并行执行、实时Token计数与成本追踪

🛠️ 43+ Tools

文件I/O、搜索、Notebook编辑、子智能体协同、任务管理、MCP接入

📚 Skills按需加载

Markdown格式技能文件，放入skills/目录自动加载

🧠 Memory

上下文自动压缩、会话恢复、断点续聊

🛡️ 权限控制

执行前权限检查、敏感操作确认、危险命令拦截

👥 Multi-Agent

子Agent生成、团队注册、后台任务管理、并行执行
