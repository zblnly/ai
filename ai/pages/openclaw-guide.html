⚡ 快速开始

前置要求

  • Node.js 18+
  • 任意LLM API Key（Claude、OpenAI、或本地Ollama）
  • （可选）微信/飞书/钉钉账号用于IM接入

安装

git clone https://github.com/openclaw/openclaw.git cd openclaw npm install cp .env.example .env # 编辑 .env 填入 API Key

配置Claude API

# .env ANTHROPIC_API_KEY=sk-ant-xxx DEFAULT_MODEL=claude-opus-4-7

配置本地Ollama（免费方案）

# .env OLLAMA_BASE_URL=http://localhost:11434/v1 OPENAI_API_KEY=ollama DEFAULT_MODEL=qwen3:32b

启动

npm start # 命令行模式 npm run im # IM接入模式（微信/飞书/钉钉） npm run server # Web UI模式

💡 最佳实践

  • 渐进式开放权限: 先让Agent只读，观察1-2周后再开放写入权限
  • 明确Skill边界: 每个Skill只做一件事，组合使用而非单个万能Skill
  • 日志全记录: 开启完整审计日志，便于问题排查和效果评估
  • 定期更新Prompt: 根据使用反馈每2周优化一次系统Prompt
  • 本地模型+云端模型混合: 敏感任务用本地模型，复杂任务用云端模型
