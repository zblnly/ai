🪼 OpenHarness 完全指南

港大开源轻量级 AI 智能体驾驭框架 | 8.8K+ GitHub Stars

什么是 OpenHarness？

OpenHarness（简称 oh）是由香港大学数据科学研究院（HKUDS）开源的轻量级 AI 智能体驾驭框架。核心定位是为大语言模型（LLM）提供完整的运行时基础设施，将普通 LLM 快速转化为具备工具调用、记忆管理、安全治理、多智能体协同能力的全能 AI 助手。

核心亮点

  • 📦 极致轻量化：仅用 1.17 万行 Python 代码实现 Claude Code 约 98% 的工具覆盖能力
  • 🔌 模型中立：支持任意大模型（OpenAI、Claude、DeepSeek、通义千问、Ollama 本地模型等）
  • 🔧 43+ 工具：文件 IO、Shell 执行、Web 搜索、数据爬取、图表生成、MCP 协议等
  • 🛡️ 安全治理：多级权限控制、Git 自动提交、一键撤销
  • 🤝 多智能体：subagent 协调、团队协作、后台任务执行
  • 💾 持久记忆：MEMORY.md 持久化、session resume、auto-compact

快速开始

1. 安装 OpenHarness

# 一键安装（推荐） curl -fsSL https://raw.githubusercontent.com/HKUDS/OpenHarness/main/scripts/install.sh | bash # 或使用 pip 安装 pip install openharness-ai # 或使用 npm npm install -g openharness-ai # 验证安装 oh --version

2. 配置模型

# 交互式配置向导 oh setup # 选择你的 provider：Claude / OpenAI / Ollama / DeepSeek / 通义千问等

3. 快速使用

# 交互式对话模式 oh # 单次提问模式 oh -p "解释这个代码库"

常用命令

斜杠命令

  • /diff - 查看当前更改
  • /undo - 撤销上一步操作
  • /commit - 提交到 Git
  • /plan - 计划模式（只读）
  • /review - 代码审查
  • /cost - 查看使用成本
  • /compact - 压缩上下文

ohmo 个人助手

ohmo 是基于 OpenHarness 的个人 AI 助手应用：

# 初始化 ohmo ohmo init # 查看配置 ohmo config # 启动个人助手 ohmo # 查看 gateway 状态 ohmo gateway status

Git 集成功能

OpenHarness 内置强大的 Git 安全保护：

  • 自动提交：AI 的每一次编辑都会自动提交到 Git
  • 一键撤销：使用 /undo 立即撤销任何更改
  • 差异查看：使用 /diff 查看变更
  • 代码审查：使用 /review 进行代码审查

权限控制

OpenHarness 提供多级权限模式：

  • Read-only：仅读取文件，不能修改
  • Auto-edit：自动编辑，需确认危险操作
  • Full：完全权限，可执行任意命令
  • Interactive：交互式确认每一步操作

本地离线模式

使用 Ollama 实现完全本地离线的 AI 编程：

# 确保 Ollama 正在运行 ollama serve # 使用本地模型 oh --model ollama/llama3 oh --model ollama/qwen2.5:7b oh --model ollama/codellama:7b

多智能体协同

OpenHarness 支持多智能体协调工作：

  • Subagent：可以派生子任务代理
  • Team：注册多个智能体组成团队
  • Background：后台任务执行
  • Swarm：智能体群协同
