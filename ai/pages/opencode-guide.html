📖 OpenCode 是什么？

OpenCode是一个开源的AI编码代理，运行在终端、IDE或桌面端。它不仅仅是一个聊天机器人，而是一个可以执行命令、搜索代码、调试错误和修改文件的智能编程伙伴。

🚀 快速开始

1. 一键安装

# 推荐方式（Linux/macOS） curl -fsSL https://opencode.ai/install | bash # npm安装 npm install -g opencode-ai # Homebrew brew install anomalyco/tap/opencode # Windows Scoop scoop install opencode

2. 首次配置

# 运行并选择模型 opencode # 或使用 /connect 命令连接模型 /connect

3. 初始化项目（重要！）

首次使用运行 /init 让OpenCode分析你的项目：

/init # 这会生成 AGENTS.md 文件 # 建议提交到 Git，让AI理解项目规范

💡 核心工作流：Plan + Build

OpenCode独特的双模式工作流：

📋 Plan 模式

只读分析，不修改代码

按 Tab 键切换

🔨 Build 模式

全权限，执行修改

默认模式

先用Plan模式分析，确认后再切换到Build模式执行——这是专业用法。

🤖 Agents 配置（高级）

通过创建 AGENTS.md 为项目配置专属AI角色：

# 在项目根目录创建 AGENTS.md # 示例：TypeScript项目规则 --- description: SST v3 Monorepo Project --- ## 技术栈 - TypeScript + Bun - SST v3框架 ## 代码规范 - 使用TypeScript strict mode - 共享代码放 packages/core/ - 单元测试放 __tests__/ ## 项目结构 - frontend/ - React前端 - api/ - API服务 - jobs/ - 后台任务

将AGENTS.md提交到Git，整个团队都能受益。

⚙️ VSCode/IDE 中使用

  1. 在VSCode中打开终端
  2. 运行 opencode
  3. 按 Cmd+Esc（Mac）或 Ctrl+Esc（Win）在分屏中打开

⚠️ 常见问题

Q: 安装后无法运行？
A: 确保 ~/.local/bin 在PATH中，或使用完整路径 ~/.local/bin/opencode
Q: Claude模型不能用？
A: 2026年Anthropic已阻止OpenCode使用Claude，请使用GPT或Gemini。
Q: 如何完全离线使用？
A: 配置Ollama本地模型，参考上面的本地模型配置。
Q: AGENTS.md是什么？
A: 项目规则文件，让AI理解你的代码规范和架构，相当于给AI的"入职培训"。
